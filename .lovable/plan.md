

## St. Augustine Cursillo — Modern Redesign

A warm & inviting multi-page redesign of the Cursillos in Christianity website for the Diocese of St. Augustine.

### Color & Design System
- Warm gold/amber primary color with deep navy accents — evoking faith and warmth
- Cream/warm white backgrounds, soft rounded corners, gentle shadows
- Clean modern typography (system fonts) replacing the dated script fonts
- Subtle cross/dove iconography using Lucide icons

### Pages

**1. Home Page**
- Hero section with a warm gradient banner featuring "Cursillos in Christianity — Diocese of St. Augustine" and a welcoming tagline
- "What is Cursillo?" intro section with a CTA button to learn more
- Upcoming Weekends highlight cards (Men's #139, Women's #137)
- Quick links to Application and upcoming events

**2. Events Page**
- Two tabbed sections: English & Spanish
- Clean event cards showing each Ultreya/School of Leaders with location, day/time, and map pin icon
- Upcoming Cursillo Weekends section with date cards
- 4th Day/Welcome Ultreya info

**3. Newsletters Page**
- Grid of newsletter cards with dates
- Placeholder content (links to original site's newsletters)

**4. Media Page**
- Gallery/video section layout
- Embedded content placeholders

**5. Documents Page**
- List of downloadable documents with icons
- Application link prominently featured

**6. Secretariat Page**
- Leadership info and contact details
- Clean card layout for each member/role

### Shared Layout
- Modern sticky top navigation bar with logo, page links, and "De Colores!" tagline
- Mobile-responsive hamburger menu
- Footer with contact info, social links, and copyright
- Smooth page transitions

### Content
All content from the original site will be hardcoded (events, locations, dates, weekends) as static data so the site works immediately without a backend.
